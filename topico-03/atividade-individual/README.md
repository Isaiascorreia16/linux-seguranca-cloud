# Atividade prática individual - Tópico 3

## Nível realizado
Nível 2 - Intermédio

## Objetivo
Criar e publicar um serviço web em ambiente Linux usando Nginx.

## Ambiente utilizado
VM local Ubuntu, acedida via SSH com PuTTY.

## Rota de publicação
Nginx

## Ficheiros criados
- index.html
- sobre.html
- style.css

## URLs testados
- http://192.168.88.39/topico-03/index.html
- http://192.168.88.39/topico-03/sobre.html

## Evidências produzidas
- Print do browser com página publicada
- Output do curl

## Dificuldades encontradas
Porta 80 bloqueada no firewall — resolvida com ufw allow 80/tcp.
